# Lighthouse 성능 측정 기록

> 도구: Lighthouse 13.0.2 / Desktop 기준
> 🟢 좋음 (90~100) | 🟡 보통 (50~89) | 🔴 나쁨 (0~49)

---

## 지표 설명

| 지표 | 뜻 | 기준 |
|------|-----|------|
| **FCP** | 화면에 뭔가 처음 보이기까지 | 1s 이하 좋음 |
| **LCP** | 메인 콘텐츠가 완전히 보이기까지 | 2.5s 이하 좋음 |
| **TBT** | 버튼 눌렀는데 반응 없는 시간 합계 | 200ms 이하 좋음 |
| **CLS** | 스크롤 중 레이아웃이 갑자기 튀는 정도 | 0.1 이하 좋음 |
| **Speed Index** | 전체 화면이 채워지는 속도 | 3.4s 이하 좋음 |
