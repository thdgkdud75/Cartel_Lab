### 5.1 `table_name`

| 컬럼명 | 타입 | NULL | 기본값 | 제약조건 | 설명 |
|---|---|---|---|---|---|
| id | bigint | N | auto | PK | 식별자 |
| created_at | datetime | N | now() |  | 생성일 |
| updated_at | datetime | N | now() |  | 수정일 |

인덱스:
- `idx_table_name_xxx` (컬럼: xxx, 목적: 조회 성능)

관계:
- FK: `child_table.parent_id -> parent_table.id` (`ON DELETE RESTRICT/CASCADE`)

- 대량 조회되는 조건 컬럼은 인덱스 필수
